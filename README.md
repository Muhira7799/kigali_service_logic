# Kigali Service Logic
A simple TypeScript project I built to practice Object-Oriented Programming (OOP) and Interfaces. 

### Why I built this:
I wanted to apply what I'm learning in my Software Engineering degree at UNILAK into a practical script. This project simulates a service system (like tech consulting) using TypeScript to keep the data structured and clean.

### What's inside:
- **Interfaces:** To define how a service provider should look.
- **Classes:** To handle the logic of different services and pricing.
- **TypeScript:** Used strictly to avoid common bugs.

It's a work in progress as I keep exploring more advanced patterns.
